# Using the screen command

## Start all; current window

`ngrok start --all`

## Start all; current window

`screen -dmS ngrok ngrok start --all`

## Re-attach to the screen

`screen -r`

## List screen session IDs

`screen -ls`

## Re-attach by ID

`screen -r 10835`

## Detach from screen

<kbd>CTRL</kbd> + <kbd>A</kbd> >> <kbd>D</kbd>
